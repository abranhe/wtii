# Install

```
$ npm install -g wtii
```

# Usage

```console
What time is it? ⏰

Usage
  $ wtii

  ╭───────╮
  │3:07 PM│
  ╰───────╯

Options
  -f, --feedback  Send a feedback
  -h, --help      Show help message and close
  -v, --version   View package Version

Examples
  $ what-time-is-it

  ╭───────╮
  │7:03 PM│
  ╰───────╯
```
